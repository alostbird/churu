# 出入管理小程序
可以通过扫描小程序生成的小程序二维码，方便信息登记，减少纸质登记，减少人员接触，方便疫情防控。

小程序使用了微信的**云开发**，无需单独部署后端服务器，节省使用成本降低开发难度。
